ant_pak.graphic.geometry.display_lines(lines, type, color=None, scale=1)[源代码]¶

This function displays lines as circular meshes in Rhino.

Arguments: - lines: A list of line objects to be displayed. - type: Type of line display ("x" or "y"). - color: Color of the meshes (optional, default is gray). - scale: The scaling factor for the mesh size (optional, default is 1).

Returns: A circular mesh object representing the displayed lines.
