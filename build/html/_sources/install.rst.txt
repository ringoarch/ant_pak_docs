安装方法
==================

- 首先将仓库代码下载到本地，解压缩后，将文件夹放到本机`c:\Users\xxxx\AppData\Roaming\McNeel\Rhinoceros\7.0\scripts\`，其中`xxxx`为你的用户名。
- 将仓库中的UserObjects的ghuser文件，全部拷贝到`c:\Users\xxxx\AppData\Roaming\Grasshopper\UserObjects\`，其中`xxxx`为你的用户名。
- 重新的开Rhino。

插件使用方法
==================

- 使用模板，在Rhino中新建一个文件，然后打开Template文件夹中的03 Airpak Model.gh文件，可以参考案例文件进行建模。
- 模拟按照建模-转换为objects-设置-集成模型-模型设置-模拟-后处理流程来搭建电池组。
