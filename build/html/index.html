Ant pak 文档¶

这是关于Ant pak的文档，包括安装、使用、API等内容。

本插件基于Rhino和Grasshopper开发，支持在Rhino中对airpak建模、设置，并调用paraview进行可视化后处理。本插件基于如下环境测试：

Windows 10/11 Rhino 7 Grasshopper 1.0 Aiaprk 3.0.16 Paraview 5.10.1 Python 3.10.11
